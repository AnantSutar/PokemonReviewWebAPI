Pokémon Review API

ASP.NET Core Web API for Managing Pokémon, Reviewers & Reviews

A clean, fully structured backend API built with ASP.NET Core Web API, showcasing real-world backend architecture, entity relationships, and end-to-end data flow. The project includes database modeling, business logic, repositories, services, and REST endpoints designed with maintainability and scalability in mind.

Features

Pokémon Management
Create, read, update, and delete Pokémon entries with related categories and owner information.

Reviews System
Add and manage reviews for each Pokémon, including reviewer details and ratings.

Entity Relationships
Supports one-to-many and many-to-many relationships across Pokémon, Reviewers, Reviews, Categories, and Owners.

Structured Project Architecture
Clearly separated layers for:

Data access

Business logic

Controllers (REST endpoints)

DTO mapping and response handling

Clean API Responses
Consistent response types, validation, and mapping via DTOs.

Tech Stack

ASP.NET Core Web API (.NET 8)

Entity Framework Core
